# Quiz service

The quiz service is used to get, create, modify, or delete quizzes. 
By providing questions and answers, along with quiz settings and meta information, quizzes can be created inside the quiz service and stored in the connected quiz database.
The service provides many different quiz types such as Single-Choice, Multiple-Choice and Numeric.

The service provides also the possibility to solve a quiz as a TurboQuiz which includes time measuring for solving it and a certain amount of allowed mistakes.
